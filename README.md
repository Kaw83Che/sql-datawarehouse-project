# Data Warehouse and Analytics Project

## Project Overview

This project demonstrates an end-to-end **Data Warehousing and Analytics solution** using SQL Server. It covers the process of importing raw data, cleaning and integrating datasets, building a data warehouse, and generating business insights through SQL-based analytics.

The project is designed to showcase practical skills in **Data Engineering**, **Data Warehousing**, **Data Modeling**, and **Data Analytics**.

---

## Objective

Build a modern data warehouse to consolidate sales data from multiple source systems and support analytical reporting for informed business decision-making.

---

## Data Sources

The project uses CSV files from two source systems:

* ERP
* CRM

---

## Project Scope

* Import data into SQL Server
* Clean and resolve data quality issues
* Integrate ERP and CRM data into a single analytical model
* Focus on the latest dataset only
* Create clear documentation for business and analytics users

---

## Analytics Focus

SQL-based analytics are developed to generate insights into:

* Customer Behavior
* Product Performance
* Sales Trends

These insights help stakeholders understand business performance and make data-driven decisions.
